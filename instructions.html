Navigating the Homepage an Searching

The homepage for this site allows you to view the videos that have been uploaded in a table going down the page. If you would like to search for a specific video, enter your search terms in the top bar of the page and click the search button.

Uploading Files

In order to upload your own file, click on the "Let's Go!!" button on the homepage, from here you can select a file, add a description to your video, and click the "Upload" button in order to add your file to the database. (Note that this is currently non-functional, as pausch does not support the file upload portion. This issue has been noted by Dr. Xaio and he is working to resolve it.)

Getting Back to the Homepage

At any time, you can click on the "Return to Homepage" button in order to return back to the main page of this website
